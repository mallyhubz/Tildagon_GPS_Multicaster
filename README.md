# Tildagon_GPS_Multicaster
Tildagon App to Multicast your GPS location.

This badge app will use the GPS hexpansion and send your Badge Name and Lat Long out over Multicast.

There is an node.js app to receive locations and plot them on the offical EMF Map
https://github.com/mallyhubz/EMF_GPS_Multicaster_Map_Webserver

If you dont want your location broadcasted to the Camp, dont use the App !
